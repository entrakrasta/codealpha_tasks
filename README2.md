# Unemployment Analysis with Python

## Description
This project uses **Python** to study the unemployment rate in India during **2020**. It looks at how **COVID-19** affected unemployment. The project includes data cleaning, data analysis, and charts.

## Project Goals
- Load the unemployment dataset (Unemployment.csv)
- Clean the data (column names and dates)
- Find basic statistics (average, minimum, and maximum)
- Compare unemployment before and during COVID-19
- Find the states with the highest unemployment
- Show unemployment trends with charts
- Give conclusions that can help economic decisions

## Technologies Used
- Python 3.x
- pandas
- matplotlib

## Dataset
- **267 records** with monthly data (January–October 2020)
- **Main columns:**
  - Region
  - Date
  - Estimated Unemployment Rate (%)
  - Estimated Employed
  - Estimated Labour Participation Rate (%)
- **Period:** Before and during the COVID-19 pandemic

## Results
The average unemployment rate in India increased from **9.76%** before COVID-19 to **13.28%** during COVID-19. This is an increase of about **36%**.

### States with the Highest Average Unemployment
- Haryana: **27.5%**
- Tripura: **25.1%**
- Jharkhand: **19.5%**
- Bihar: **19.5%**
- Delhi: **18.4%**

The highest unemployment was in **April and May 2020**, during the national lockdown.
